Third Party System Interface

Chillers and Primary Chilled Water Pumps

BMS will not have any control on the operation of the chillers and primary chilled water pumps system. Chillers supplying vendor (Carrier) will monitor and control the entire system and will send following information to BMS through MODBUS-RTU over TCP/IP using terminal server (provided by others) for monitoring only.

  • Chiller Enable / Disable Command
  • Chillers Hun Status
  • Chillers Fault Signal
  • Local/CCN/Remote Switch Status
  • Chiller Leaving Immersion Temperature Sensor
  • Butterfly Valve Open/ Close Status
  • Common Chiller Supply Immersion Temperature Sensor
  • Pump Start/Stop
  • Pump Run Status
  • Pump Trip Signal
  • Pump Hand/Off/Auto Signal

UPS Monitoring

BMS shall be monitoring the following points through MODBUS-RTU over TCP/IP using terminal server (provided by others) for UPS System.

  • System Healthy Status
  • Common Fault
  • Battery Status

Generator

BMS shall be monitoring the following points through MODBUS-RTU over TCP/IP using terminal server (provided by others) for Generator System.

  • System Healthy Status (Auto Position)
  • Run Status
  • Common Fault Signal(Shut Down, Warning, Electrical Trip)
  • Emergency Stop Status
  • Hi Coolant Temp Alarm Signal
  • Generator On Test Status
  • Battery Status
  • Fuel Tank Level
  • Voltage/Current/Frequency/Power
