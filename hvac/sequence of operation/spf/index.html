Staircase Pressurization Fan

System Overview

BMS will not have any control on the operation of the staircase pressurization fan. Fire alarm system will have primary control on the start/stop operation of the staircase pressurization fan system. Staircase pressurization fan will be stopped automatically once the Fire signal return to normal.
BMS shall only monitor the following:

  • Fan HOA Switch Status
  • Fan Run Status
  • Fan Trip status

Alarms

The following alarms would be generated on BMS.

Fan Trip:This is generated when the Fan Trip Signal is on as sensed by panel volt free contact.
