Fan Coil Unit

System Overview

The FCU controllers will be integrated to the Central BMS system via LON bus.
The fan coil units will be controlled by a dedicated unitary controller, which will be hardwired to return air duct temperature sensor.

Temperature Control

The return air temperature sensor installed in the return air duct or void, will relay the measured signal (temperature) to the FCU controller, the FCU controller compares this signal with the set-point. Based on the comparison, the DDC controller generates an analog output to the 2-way floating cooling valve. A Proportional - Integral - Derivative program will determine the percentage of the cooling coil valves opening to achieve the desired condition. The 2-way valve is mounted on the CHW return pipe. When the fan coil unit is off the chilled water 2-way valve will modulate to the full close position.
