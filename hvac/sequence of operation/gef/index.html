Garbage Exhaust Fan

System Overview

The fans are located inÂ·the roof of each block and are used to extract air from garbage room.
The unit is comprised of fixed speed extract Fan. The sequence of operation of the unit shall be as follows:

Fan Control

  1. The extract fan shall be operated by the BMS when:
    • The HOA switch of the fans is in "Auto" position.
    • The fan trip signal is in NORMAL state.
  2. Once the above conditions are satisfied, extract fan is enabled to start in Auto mode (based on time schedule) or using a plant enable button on the graphics in manual mode by operator.
  3. Once enabled BMS will automatically send fan start command to extract fan MCC panel. BMS shall monitor the differential pressure airflow switch of the fan..
  4. If the fan status from the differential pressure air flow switch is not 'On' within preset time delay of 120 seconds after the start command is being issued then an alarm will be generated and latched on the BMS and the extract fan shall be stopped.
  5. The controls shall be programmed to operate the extract fan for continuous 24 Hours, 7 days a week time schedule.
  6. The operator can stop the extract fan from the BMS anytime; however, the default state of the extract fan when the DOC is reset or powered up is START.
  7. To do the plant - stop, the manual mode check box provided in the graphics should be checked and then the Start/Stop button should be selected to STOP option. Operator should uncheck the mode checkbox to run the system in Auto mode.

Alarm Reset

When the extract fan start command is issued by the BMS, then the BMS monitors the.TRIP status of the fan & the RUN status of the fan from the differential pressure switch. In case that if any of the status goes to OFF/ALARM condition, then BMS will generate a FAN FAIL ALARM (point will be displayed on the central graphic). This alarm shall be latched until the operator resets it. The extract fan start/stop command shall be OFF from the BMS until this alarm is reset

Plant Reset

If the unit is in "Fail" condition, then the operator can reset this alarm by using the plantreset button available on graphic screen or by putting the respective equipment H/O/A switch status in OFF or HAND mode and bringing it back to AUTO mode from local control panel (This provision is provided to facilitate the operator to reset respective equipment alarms in case graphic communication fails with controllers). To do the Plant -reset, the manual mode check box provided in the graphics should be checked and then the RESET option should be selected. Once the RESET is activated it will come back to NORMAL state automatically after predefined delay time and mode checkbox will be unchecked automatically.

Alarms

The following alarms would be generated on BMS.

Extract Fan Trip:This is generated when the fan motor is tripped as sensed by panel volt free contact.
The above alarms are active on the BMS irrespective of whether the H/O/A switch is in "Hand" or "Auto central" position.
Extract Fan Fail:In case the extract fan fails to start or if the run status from the differential pressure switch is not giving the signal according to the fan start command (given from the BMS) due to any reason or the fan is giving the trip status then an alarm will be generated.
